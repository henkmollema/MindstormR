MindstormR Controller
URL Uitleg Response type
/login Logt de robot in op de webserver. Geeft een ID terug. int
/{id}/logout Logt de robot met het opgegeven id uit. bool
/all Geeft een lijst van alle ingelogde robots terug. JSON object
/flush Logt alle robots uit op de webserver. bool
/{id}/{command} Slaat het opgegeven command op in de wachtrij van de robot met het opgegeven id. bool
/{id}/command
?touch={touch}&gyro={gyro}
&color={color}&ir={ir}&battery={battery} 		Haalt het eerstvolgende op voor de robot met het opgegeven id. De meegstuurde sensor data wordt opgeslagen. JSON object
/{id}/sensors/get Geeft de sensor data van de robot met het opgegeven id terug. JSON object
